# 🚀 START HERE - Your Rackspace Chatbot

## 🎯 What This Is

A **complete AI chatbot system** where YOU:
1. **Collect data** using BFS web crawling (200-500+ docs from all Rackspace domains)
2. **Train YOUR OWN model** (fine-tune TinyLlama on your data)
3. **Deploy locally** (no agents, no APIs, full control)

## ⚡ Quick Start (3 Commands)

```bash
# 1. Setup (5-10 min)
./setup.sh

# 2. Build everything (1-2 hours)
source venv/bin/activate
./build_pipeline.sh

# 3. Launch chatbot
python app.py
```

Then open: **http://localhost:7860**
